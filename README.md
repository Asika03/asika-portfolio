# Asika A — Personal Portfolio

## Setup

1. **Add your photo:**
   - Save your profile photo as `assets/asika.jpg` inside this folder
   - The photo frame is 280×320px — portrait orientation works best

2. **Open the portfolio:**
   - Simply open `index.html` in any browser — no build step needed
   - Or use VS Code Live Server for best experience

## Structure
```
asika-portfolio/
├── index.html      # All sections (Hero, About, Skills, Experience, Projects, Achievements, Contact)
├── style.css       # All styles (dark navy, glassmorphism, aurora gradients)
├── script.js       # Navigation, typewriter, reveal animations
└── assets/
    └── asika.jpg   ← Place your photo here
```

## Sections
- **Hero** — Name, animated roles, CTA buttons
- **About** — Bio, photo, beyond coding
- **Skills** — Categorized skill boxes
- **Experience** — Timeline (Aivar, CogniTech, InternEzy)
- **Projects** — 6 case-study cards + featured LLM project
- **Achievements** — Certifications + leadership roles
- **Contact** — Links + contact form
